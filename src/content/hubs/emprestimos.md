# Guia Completo: Empréstimos

Às vezes, você precisa de dinheiro agora. A pergunta é: **qual é a melhor forma de conseguir?**

Este hub reúne **tudo que você precisa saber** sobre empréstimos para tomar decisões inteligentes.

---

## Comece Aqui

### Tipos de Empréstimo (Do Melhor para o Pior)

1. **Empréstimo Consignado** - Juros baixos, risco controlado.
2. **Crédito Pessoal** - Acessível, mas juros mais altos.
3. **Cheque Especial** - Disponível mas muito caro.
4. **Cartão Rotativo** - Evite. A pior opção possível.

### Antes de Pedir Emprestado

- ✓ Isso é realmente necessário AGORA?
- ✓ Quanto vou pagar de juros?
- ✓ Consigo pagar as parcelas?
- ✓ Explorei alternativas (FGTS, empréstimo com família)?

## Conceitos-Chave

**Juros:** O preço de pedir dinheiro emprestado.

**Taxa Nominal vs Taxa Real:** Nominal é o que falam, real inclui tudo (IOF, etc).

**Parcela Fixa:** Mesma quantia todo mês (padrão em empréstimos).

**Amortização:** Quanto da parcela reduz a dívida vs quanto é juros.

**Portabilidade:** Direito de transferir crédito para instituição com taxas menores.
