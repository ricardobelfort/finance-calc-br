# Planilha ou App: Qual Ferramenta de Orçamento?

Você precisa rastrear gastos.

Opções:
1. Excel / Google Sheets
2. App especializado (Organizze, Guiabolso, etc)
3. Papel e caneta

Qual escolher?

---

## Opção 1: Google Sheets (Gratuito)

### Vantagens

✓ Gratuito
✓ Acessa de qualquer lugar
✓ Customizável (faz do seu jeito)
✓ Sincroniza automaticamente
✓ Versão móvel decente

### Desvantagens

✗ Exige disciplina (insere manualmente)
✗ Sem categorização automática
✗ Sem integração com banco
✗ Curva de aprendizado

### Melhor Para

Quem gosta de controlar tudo, não tem pressa, quer customizar.

### Modelo Básico

```
Data | Descrição | Categoria | Valor | Tipo
30/01 | Almoço | Alimentação | 35 | Gasto
30/01 | Salário | Entrada | 3000 | Entrada

Resumo por categoria:
Alimentação: R$350
Transporte: R$120
...
```

---

## Opção 2: Excel

Similar ao Google Sheets.

**Melhor:** Offline, mais poderoso para fórmulas.

**Pior:** Precisa sincronizar manual, não acessa no celular fácil.

**Verdict:** Escolhe Google Sheets (é grátis e melhor).

---

## Opção 3: Organizze

App popular no Brasil.

### Vantagens

✓ Sincroniza com banco (alguns)
✓ Categorização automática
✓ Interface amigável
✓ Versão web + mobile
✓ Relatórios legais

### Desvantagens

✗ Pago (R$20-30/mês ou R$200/ano)
✗ Conta integração com banco nem sempre funciona
✗ Vêndedor de seguros (oferece seguros, pode ficar incômodo)

### Melhor Para

Quem quer algo pronto, usa app no celular, não se importa pagar pouco.

### Preço

Lite: Grátis (básico)
Premium: R$299/ano ou R$29.90/mês

---

## Opção 4: Guiabolso

Outro app popular.

### Vantagens

✓ Sincroniza com maioria dos bancos
✓ Importa transações automaticamente
✓ Interface mobile ótima
✓ Freemium (usa sem pagar)

### Desvantagens

✗ Versão paga é cara (R$20-30/mês)
✗ Coleta dados para vender (banco de dados)
✗ Às vezes bugs

### Melhor Para

Quem quer automático, não se importa com privacidade, gosta de mobile.

---

## Opção 5: Nubank App

Seu próprio banco oferece.

### Vantagens

✓ Integrado com sua conta Nubank
✓ Transações automáticas
✓ Gratuito
✓ Interface bonita

### Desvantagens

✗ Só funciona para Nubank
✗ Menos customizável
✗ Não integra outros bancos

### Melhor Para

Se usa Nubank como banco principal.

---

## Opção 6: Papel e Caneta

Sim, existe.

### Vantagens

✓ Nenhuma tecnologia
✓ Você escreve, memoriza melhor
✓ Gratuito

### Desvantagens

✗ Sem cálculos automáticos
✗ Fácil errar
✗ Difícil visualizar padrões
✗ Perde anotações

### Melhor Para

Ninguém realmente. Mas algumas pessoas fazem por ser tátil.

## Comparação Rápida

| Ferramenta | Custo | Automático? | Celular? | Melhor Para |
|-----------|-------|-----------|---------|-----------|
| Google Sheets | Grátis | Não | Sim | Controlar tudo |
| Organizze | Pago | Parcial | Sim | Interface boa |
| Guiabolso | Grátis/Pago | Sim | Sim | Preguiçoso |
| Nubank App | Grátis | Sim | Sim | Usuário Nubank |

## Minha Recomendação

### Se Quer Grátis

Google Sheets.

Simples, funciona, customizável.

### Se Quer Automático

Guiabolso versão grátis.

Integra com banco, não precisa pensar.

### Se Pode Pagar

Organizze Premium ou Guiabolso Premium.

Valem a pena (R$20-30/mês = R$2-3 por dia).

---

## Dica: Comece Simples

Não escolhe a ferramenta "perfeita".

Escolhe UMA e **consistentemente** rastreia por 3 meses.

Após 3 meses de dados, decide se continua ou troca.

---

## FAQ

**P: Qual é mais seguro?**

R: Google Sheets (você controla). Apps compartilham dados.

**P: Qual tem melhor relatório?**

R: Organizze e Guiabolso têm relatórios bonitinhos.

**P: Posso usar 2 simultaneamente?**

R: Pode, mas complica. Escolhe 1.

**P: E se não usar por 1 mês?**

R: Ferramenta nenhuma ajuda. O problema é disciplina, não a ferramenta.
